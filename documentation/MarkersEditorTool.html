Markers Editor

This tool is very simple to use, the user only has to select the type of marker that better fits to his needs, and put his position.

 

Available markers:

      Start: All stream sounds and musics must have at least one. In the music file, these markers are also used for creating jump points.

      End: They stop playback when reaches the specified position, this one is not needed in files that contains loop points.

      Goto: When the player reaches the specified position, returns to the previous specified. This one is used for musics and unused in the stream file, works similar as a Loop marker. The difference is that Loop Marker creates two start markers and three markers on the other hand, Goto Marker only creates one of each.

      Loop Marker: When the player reaches the specified position, returns to the previous specified, is always accompanied by its own trailing Start marker.
